Controls:
Use the buttons to control the car's movement.
Use W, A, S, D keys for manual movement:
Forward: w
Backward: s
Left: a
Right: d
Use I, J, K, L keys for controlling the camera mount:
Increase speed: i
Decrease speed: k
Turn left: j
Turn right: l
Stop movement: spacebar
Car needs to be "started" to use manual controls.
